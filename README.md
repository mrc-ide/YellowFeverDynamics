# YellowFeverDynamics
A package for running dynamic SEIRV models of yellow fever, auxiliary to the [YEP (Yellow Fever Epidemic Prevention) package](https://github.com/mrc-ide/YEP). This is the in-progress version 0.3 of YellowFeverDynamics, which makes use of the updates to the standard yellow fever model in version 0.3 of YEP.
